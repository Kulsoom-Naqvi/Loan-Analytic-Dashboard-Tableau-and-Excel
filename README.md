
# 📊 Loan Analytics Dashboard | Tableau

> **An interactive Tableau dashboard that transforms raw banking loan data into actionable insights by analyzing lending performance, borrower demographics, portfolio health, repayment behavior, and credit risk—all within a single dynamic dashboard.**

---

## 🚀 Project Overview

Banks generate thousands of loan records every day, making it difficult to monitor portfolio performance, repayment trends, customer demographics, and financial risk through raw data alone.

This **Loan Analytics Dashboard** was developed to bridge that gap by converting complex lending data into an intuitive, interactive dashboard. Instead of scrolling through spreadsheets, decision-makers can instantly evaluate the health of the loan portfolio, identify risk areas, compare regional performance, and understand borrower behavior through meaningful visualizations.

The dashboard combines **financial KPIs, geographic analysis, customer segmentation, repayment monitoring, and portfolio quality** into one centralized analytical solution, enabling faster and more informed business decisions.

---

# 🎯 Business Problem

Financial institutions need to answer critical questions every day:

* How much loan amount has been disbursed?
* Which branches are generating the highest revenue?
* Which states have the highest lending activity?
* Which loan products contribute the most?
* How healthy is the current loan portfolio?
* Are delinquent and default loans increasing?
* Which customer segments require greater attention?

This dashboard provides immediate answers through interactive analytics.

# ⚙️ Interactive Features

One of the key strengths of this dashboard is its interactivity.

### 🔹 Interactive Filters

Users can dynamically filter the dashboard using different business dimensions, including:

* Branch
* State
* Loan Status
* Grade
* Religion
* Age Group

Every visualization updates instantly based on the selected filters, making the dashboard highly exploratory.

---

### 🔹 Hover Tooltips
To improve readability without overcrowding the dashboard, hover tooltips have been implemented.
The following visualizations provide additional information on hover:

* Religion-Wise Loan Distribution
* Age Group-Wise Loan Distribution

These tooltips reveal detailed values and contextual information while keeping the dashboard clean and visually appealing.

---

### 🔹 Dynamic Parameter Selection

A **Select Metric** parameter enables users to switch between different financial metrics:

* Interest
* Fees
* Total Revenue

This allows multiple business perspectives to be analyzed without creating separate dashboards.

# 📈 What This Dashboard Tells

Rather than displaying isolated charts, the dashboard tells the complete story of a bank's lending operations.

It answers four fundamental questions:

### 💰 How is the business performing?

Through loan amount, collections, interest earned, and revenue generation.

### 🌍 Where is lending happening?

Through branch-wise and state-wise loan distribution.

### 👥 Who are the borrowers?

Through religion, age group, product preference, and borrower grade analysis.

### ⚠️ How healthy is the portfolio?

Through delinquent loans, default rates, loan status, loan maturity, and verification monitoring.

Together, these insights help financial institutions improve lending strategies, reduce risk, and monitor portfolio performance more effectively.

## 📊 Alternative Implementation in Microsoft Excel

This project has also been developed in Microsoft Excel, recreating the same analytical dashboard using Excel's data visualization and dashboarding capabilities.

The Excel version presents the same key performance indicators, business metrics, and analytical insights as the Tableau dashboard, while demonstrating how similar business intelligence solutions can be built using spreadsheet-based tools.

**The Excel dashboard includes:**
- Interactive dashboard layout
- KPI summary cards
- Charts and visualizations
- Business insights
- Dynamic filtering and analysis
- Clean and user-friendly dashboard design

Having both implementations showcases the ability to solve the same business problem using different Business Intelligence platforms, highlighting adaptability across industry-standard analytics tools.

📁 The repository includes both the Tableau Workbook (`.twbx`) and the Microsoft Excel Dashboard (`.xlsx`). Feel free to explore both versions to compare how the same loan analytics solution has been implemented using two different technologies while delivering the same business insights.

## 🌟 Project Summary

This project demonstrates the design and development of an interactive Loan Analytics Dashboard using Tableau, with an equivalent implementation in Microsoft Excel. Both dashboards analyze the same banking dataset and deliver consistent business insights through KPI tracking, interactive visualizations, demographic analysis, portfolio monitoring, and lending performance evaluation. Together, they showcase how modern Business Intelligence solutions can be built across multiple analytics platforms while maintaining the same analytical objectives and user experience.
